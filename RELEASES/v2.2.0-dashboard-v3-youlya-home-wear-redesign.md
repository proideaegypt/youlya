## What Changed
- Redesigned dashboard shell into modern YOULYA HOME WEAR branded layout with responsive sidebar/topbar/footer.
- Added bilingual UI support (`عربي / EN`) with runtime `dir`/`lang` switching and Arabic default.
- Added light/dark brand theme tokens with pink/charcoal palette and refined surface/contrast system.
- Added reusable UI components: logo, status badges, KPI cards, chart cards, empty states, and animated panels.
- Upgraded Command Center, Inbox, Orders, Logs, and Settings pages with polished structure, headings, and safer empty/data states.
- Polished login page to match dashboard branding while preserving existing auth mechanics.
- Kept auth/session guard in dashboard layout and retained BuildIdentityFooter.
- Updated dashboard Playwright expectations and task artifact routing for redesigned UI.

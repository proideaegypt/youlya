# Release: v2.20.0 — simplify-human-handoff-and-add-records-filters-exports

## Task
simplify-human-handoff-and-add-records-filters-exports

## Release Type
minor

## What Changed
- Replaced the broad human-handoff trigger logic with a strict policy that only auto-handoffs on explicit customer service requests or explicit manager/admin/responsible-person requests.
- Added handoff policy classification, configurable human-handoff settings, pause-after-handoff behavior, human-team notifications, and return-to-AI support.
- Updated the dashboard human-handoff center with problem summaries, type/priority filters, date filtering, and return/resolve/takeover actions.
- Added reusable record date filtering and export UI foundations, and wired them into the main record pages covered in this task.
- Added tests for handoff classification, pause/resume behavior, settings load/save, and message-turn handoff behavior.
- Export coverage remains partial on the more complex product/learning pages and is documented in the task report.

## What Changed
- Created Phase E internal WhatsApp + n8n pilot runbook at `docs/PHASE_E_INTERNAL_WHATSAPP_N8N_PILOT.md` with environment readiness checklist, stop conditions, and n8n/Evolution connection steps.
- Created 10-scenario manual test template at `qa-artifacts/manual-tests/2026-05-02/internal-whatsapp-n8n-pilot.md` (WA-001 through WA-010) covering greeting, product search, Latin/Arabic selection, address collection, confirmation gate, duplicate protection, handoff, unclear intent, and kill switch.
- Created optional internal pilot smoke test script at `scripts/internal-pilot-smoke.mjs` to verify public health endpoints and auth boundaries without side effects.
- Fixed `lib/ui/dashboard-sidebar.tsx` lint error by moving localStorage read from `useEffect` into `useState` initializer to avoid `react-hooks/set-state-in-effect` cascade warning.
- Fixed `lib/ui/theme-toggle.tsx` lint error by adding `eslint-disable-next-line` comment for the standard next-themes hydration guard pattern.
- Fixed `eslint.config.mjs` by adding `reference/` to `globalIgnores` to prevent linting raw theme reference files.
- Note: release also carries forward all prior uncommitted changes from v2.3.0/v2.4.0 dashboard redesign tasks.

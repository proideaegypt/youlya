## What Changed
- Added dashboard Playwright QA swarm infrastructure with dedicated projects and auth setup.
- Added environment template for Playwright dashboard QA (`.env.playwright.example`) and ignore rules for local secrets/state.
- Added auth bootstrap test `tests/playwright/auth.setup.ts` with env validation and storage-state generation.
- Added swarm suites:
  - `tests/playwright/dashboard-ux-swarm.spec.ts`
  - `tests/playwright/dashboard-functional-swarm.spec.ts`
  - `tests/playwright/dashboard-a11y-rtl-swarm.spec.ts`
  - `tests/playwright/dashboard-api-health-swarm.spec.ts`
- Added shared Playwright QA helper utilities: `tests/playwright/helpers.ts`.
- Updated `playwright.config.ts` with swarm projects, auth dependency, output directory, and report output.
- Added QA report collector script: `scripts/collect-playwright-swarm-report.mjs`.
- Added manual testing runbook and template:
  - `docs/N8N_WHATSAPP_MANUAL_TEST_RUNBOOK.md`
  - `qa-artifacts/manual-tests/2026-05-01/n8n-whatsapp-pilot-template.md`
- Added npm scripts for dashboard swarm runs and report aggregation.
- Added baseline/result QA artifacts for this task.

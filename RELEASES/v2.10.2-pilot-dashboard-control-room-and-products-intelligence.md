## What Changed
- Added `/dashboard/pilot-control` as a live dashboard route for the 48h pilot.
- Added a pilot control room UI with health, n8n workflow, Evolution, kill switch, dead-letter, handoff, duplicate-block, and last inbound/outbound message panels.
- Masked message previews in the pilot control API to avoid exposing PII in dashboard previews.
- Improved Products Intelligence to surface product photos, AI-order notes, top AI product, top channel, per-product order counts, and channel empty states for Instagram/TikTok/Facebook.
- Added a shared products-intelligence helper layer so the dashboard routes can derive safe channel/order summaries consistently.
- Wired the pilot control route into the dashboard shell navigation and command-center quick actions.
- Added Playwright coverage for pilot control, products intelligence, navigation, and dashboard API health.
- Fixed the login logo hydration mismatch so local Playwright auth setup can complete cleanly.

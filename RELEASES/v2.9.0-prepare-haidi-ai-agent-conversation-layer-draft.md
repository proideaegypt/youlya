## What Changed
- Added `docs/HAIDI_AI_SALES_AGENT_PROMPT.md` — production-ready system prompt for Haidi AI Sales Agent
- Added `lib/services/haidi-context-builder.ts` — builds safe commerce facts context for Haidi consumption
- Added `lib/services/haidi-output-validator.ts` — validates Haidi JSON output, blocks unsafe order claims, falls back to app reply
- Extended `lib/types/messages.ts` — added optional `haidi_context` to `MessageTurnResponse`
- Updated `lib/services/message-turn-service.ts` — populates `haidi_context` for all major intents (PRODUCT_SEARCH, SELECT_PRODUCT, CONFIRM_ORDER, handoff, unclear, fallback)
- Added `n8n/workflows/youlya-whatsapp-main-haidi-draft.json` — draft workflow with Haidi layer (active=false)
- Added `n8n/workflows/HAIDI_AGENT_WORKFLOW_PATCH_PLAN.md` — exact node names, connections, prompts, and credential requirements for Haidi integration
- Added `docs/HAIDI_MEMORY_DESIGN.md` — memory rules: app owns commerce truth, Haidi owns conversational warmth, no product index mapping in memory, no PII in AI memory
- Updated `scripts/validate-n8n-workflows.mjs` — validates Haidi draft workflow for inactive status, turn endpoint presence, no Shopify direct nodes, no hardcoded secrets
- Added `tests/unit/haidi-agent.test.ts` — 17 tests covering:
  - Haidi output validation (valid JSON, empty reply, unsafe order claims, fallback behavior)
  - haidi_context builder (max 10 products, product facts extraction, replyGoal inference, cart summary, blocked reason)
  - Secret exposure prevention

## Safety Rules Enforced
- Haidi cannot override app action/order status
- Haidi cannot invent prices, stock, SKUs, product IDs, variant IDs
- Haidi must use only app-provided commerce facts
- Product index resolution stays in app (last_product_recommendations)
- Order confirmation only when app action=order_created
- Draft workflow active=false
- No Shopify direct mutation nodes in draft
- No hardcoded secrets in any workflow

## Architecture
```
Evolution Webhook → n8n Guard → Normalize → Call Turn Endpoint (/api/internal/messages/turn)
  → Haidi AI Sales Agent (draft, not active)
  → Validate Haidi Output
  → Prepare Reply → Should Send Reply → Send Text
```

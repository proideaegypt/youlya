## What Changed
- Added Products & Inventory dashboard module at `/dashboard/products`
- Implemented 6 tabs: Overview, Catalog Cache, Variants, Sync Health, Search QA, Mapping Inspector
- Overview tab: KPIs (total products, variants, AI-visible, available, OOS, missing SKU), cache health score, AI sellable score, variant distribution charts, AI visibility funnel
- Catalog Cache tab: paginated product table with variant counts, availability, AI visibility, Shopify links
- Variants tab: paginated variant table with filters (all, OOS, missing SKU, AI visible, low stock), hidden reason analysis
- Sync Health tab: sync status, counts, n8n daily workflow status, manual sync instructions
- Search QA tab: internal search test form with product/variant ID inspection
- Mapping Inspector tab: paginated view of last_product_recommendations with customer ID masking and expiry status
- Added read-only dashboard API routes: overview, catalog, variants, sync-health, mapping-inspector, search-qa
- Updated dashboard sidebar with Products & Inventory navigation
- Updated all Playwright swarm specs to include `/dashboard/products` and new API routes

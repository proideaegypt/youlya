# Release: v2.4.0 — port-smart-home-theme-to-youlya-admin-dashboard

## Task
port-smart-home-theme-to-youlya-admin-dashboard

## Release Type
minor

## What Changed

### Smart Home Theme Port
- Ported polished admin dashboard UI from ThemeWagon Smart Home Next.js theme
- Replaced MUI-based dashboard with Tailwind CSS + CSS variables design system
- Youlya brand colors mapped to CSS variable tokens (primary: #EFB6C1)

### Dependencies Added
- lucide-react (icons)
- class-variance-authority, clsx, tailwind-merge (cn utility)

### New/Updated Components
- `lib/ui/theme-provider.tsx` - next-themes wrapper with class attribute
- `lib/ui/theme-toggle.tsx` - Light/dark toggle with Moon/Sun icons
- `lib/ui/language-toggle.tsx` - Arabic/English toggle with Globe icon
- `lib/ui/dashboard-sidebar.tsx` - Gradient sidebar with collapse, logo, nav, logout
- `lib/ui/dashboard-topbar.tsx` - Search, AI status, notifications, user
- `lib/ui/dashboard-shell.tsx` - Smart Home inspired shell layout
- `lib/ui/build-identity-footer.tsx` - Updated for new CSS variables
- `lib/ui/youlya-logo.tsx` - Updated for new CSS variables
- `lib/utils.ts` - cn helper utility

### Redesigned Pages
- `/login` - Smart Home card-style auth page with Youlya branding
- `/dashboard/command-center` - Welcome gradient card + KPI widgets + charts
- `/dashboard/inbox` - Message-style handoff list + conversation preview
- `/dashboard/orders` - Stats cards + filterable data table
- `/dashboard/logs` - Summary cards + filterable log entries with badges
- `/dashboard/settings` - AI controls + integration health + system status + brand preview

### CSS System
- Replaced globals.css with Smart Home CSS variable system
- Light/dark mode tokens for all surfaces, text, borders
- Brand gradient utilities
- Recharts theming support

### Playwright Updates
- Updated a11y test to allow icon buttons with aria-label
- Updated functional tests for new page headings and labels

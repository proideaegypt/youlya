## What Changed
- Updated `scripts/scan-secrets.mjs` to ignore server-local env files and deploy artifacts (`.env*`, `.env.production.backup*`, `qa-artifacts`, lock files, and build-info output) while still scanning committed app/docs/scripts/release files.
- Added `scripts/check-env-files-not-tracked.mjs` and wired `npm run check:env:tracking` to fail if sensitive env files are tracked or not ignored.
- Added `scripts/check-production-env.mjs` and wired `npm run check:env:production` to validate required production env key names only (no values printed).
- Updated `scripts/verify-before-deploy.sh` to run env-tracking and production-env checks before typecheck/lint/tests.
- Added strict TLS readiness check script `scripts/check-tls-readiness.sh` (`npm run check:tls`) without `curl -k`.
- Added TLS operations runbook `docs/TLS_CERTIFICATE_RENEWAL.md`.
- Updated README command list with new deploy-safety checks.

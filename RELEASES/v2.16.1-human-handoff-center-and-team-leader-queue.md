- Hardened handoff-service.ts with explicit `pauseAIForConversation` and DB-level `setAIPaused` integration

- Dashboard handoff API returns raw `conversationId` + masked `conversationIdDisplay` for authenticated staff
- Dashboard handoff UI added conversation timeline preview (fetches `/api/dashboard/conversations/{id}/timeline`)
- Added `aria-label` filters for a11y compliance
- Added `/dashboard/handoff` to Playwright a11y and functional swarms
- All handoff-center unit tests pass (148 total)

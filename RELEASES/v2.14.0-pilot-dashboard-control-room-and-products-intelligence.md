## What Changed
- Added `/dashboard/pilot-control` page and `/api/dashboard/pilot-control` API route for pilot monitoring.
- Pilot control shows: health status (Supabase, n8n, Evolution, Shopify), kill switch status, n8n workflow active/pending, Evolution connected/pending, dead letter count, handoff count, duplicate blocked count, safety blockers, last 10 inbound/outbound messages with PII masking.
- Added pilot-control navigation link to dashboard sidebar.
- Products Intelligence already had: product cards with photos, synced count, missing SKU, OOS, AI-visible, top ordered by AI, top channel, empty states for WhatsApp/Instagram/TikTok/Facebook.
- Updated Playwright UX swarm to include `/dashboard/pilot-control`.
- Added Playwright functional tests for pilot-control and products-intelligence pages.
- Fixed `mock-store.ts` to include `aiPausedConversations` Set required by `handoff-service.ts`.
- Updated `EmptyState` component to support optional `compact` prop.

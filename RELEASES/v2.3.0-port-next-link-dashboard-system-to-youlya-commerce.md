## What Changed

### Dashboard Design System Port (Next-Link Reference)
- Ported high-quality dashboard system from Next-Link as design reference
- Added MUI Material + Icons + Recharts dependencies for polished UI components
- Created MUI theme provider with Youlya brand colors (blush pink #EFB6C1, charcoal #2B2525)
- Implemented RTL-ready Arabic/English i18n dictionary

### New Components
- `lib/ui/theme-provider.tsx` - MUI theme with Youlya brand integration
- `lib/ui/dashboard-i18n.ts` - Arabic/English translation dictionary
- `lib/ui/kpi-card.tsx` - Enhanced KPI card with Recharts sparklines and progress bars
- `lib/ui/section-card.tsx` - Task list card for alerts and action items
- `lib/ui/dashboard-charts.tsx` - Recharts-based analytics charts (Area, Bar, Line, Pie)

### Updated Components
- `lib/ui/dashboard-shell.tsx` - Integrated MUI theme provider
- `lib/ui/dashboard-sidebar.tsx` - Added MUI icons, collapse animation, gradient promo card
- `lib/ui/dashboard-topbar.tsx` - Added notification bell with indicator

### Redesigned Pages
- `/dashboard/command-center` - Enhanced with sparklines, progress bars, 4-chart analytics grid, action center alerts
- `/dashboard/inbox` - Improved layout with section-card for handoffs, better empty states
- `/dashboard/orders` - Added search/filter, full table view with status badges
- `/dashboard/logs` - Added level/type filters, search, status summary badges
- `/dashboard/settings` - Added system status section, brand color preview cards

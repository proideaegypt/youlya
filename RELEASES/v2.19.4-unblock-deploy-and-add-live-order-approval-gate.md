## What Changed
- Added a hard live-order approval gate in the Shopify order service.
- Live order creation now blocks unless `OWNER_APPROVES_LIVE_ORDER=true`, while `testMode=true` and `MOCK_MODE=true` still use mock-only behavior.
- Added a safe owner-approval response path with audit logging and dashboard visibility in pilot control safety blockers.
- Fixed internal message-turn normalization so missing `conversation_id` can fall back to `remote_jid` or `phone`, and reject safely with 400 when no fallback exists.
- Fixed React lint errors in Haidi Lab and Haidi Learning pages by removing setState-in-effect patterns.
- Added regression tests for approval blocking, live approval allowlist, mock-only behavior, and conversation ID fallback.
